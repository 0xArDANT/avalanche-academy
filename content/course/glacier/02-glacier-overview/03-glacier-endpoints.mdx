### EVM Endpoints

<Accordions>
<Accordion title="Chains">
- [List All Chains](https://glacier.docs.avacloud.io/reference/supportedchains)
```bash
curl --request GET \
     --url https://glacier-api.avax.network/v1/chains \
     --header 'accept: application/json'
```
- [Get Chain Information](https://glacier.docs.avacloud.io/reference/getchaininfo)
```bash
curl --request GET \
     --url https://glacier-api.avax.network/v1/chains/{chainId} \
     --header 'accept: application/json'
```
</Accordion>
