## Prerequisites

### Glacier API Key

You will need to [create an AvaCloud account](https://app.avacloud.io) to access the Glacier API.

Once your account is created, generate a new API key by [visiting the "Web3 Data API" tab](https://app.avacloud.io/glacier-api/) and clicking "+ Add API Key."

### Software Development

A general understanding of software development is required. While you won't need to write a lot of code, familiarity with key concepts is important. We recommend:

- **TypeScript:** Understanding of common concepts, as all exercises will use TypeScript.

## Learning Outcomes

By the end of this course, you will:

- Be familiar with Glacier and its offerings
- Learn how to use the [AvaCloudSDK](https://github.com/ava-labs/avacloud-sdk-typescript) to interact with Glacier
- Build multiple web apps using the [AvaCloudSDK](https://github.com/ava-labs/avacloud-sdk-typescript)

This course will provide you with the advanced knowledge needed to build blockchain applications using Glacier.
