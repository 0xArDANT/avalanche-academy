Alright, now let's create a configuration file for the AWM Relayer.

## Get the Sample Relayer Config

You can get the sample relayer config from the [AWM Relayer GitHub repository](https://github.com/ava-labs/awm-relayer/blob/main/sample-relayer-config.json).

## General Config

We will use the local node running on port 9650 for both the P-Chain and Info API.

```json
{ 
  "p-chain-api": {
    "base-url": "http://127.0.0.1:9650",
  },
  "info-api": {
    "base-url": "http://127.0.0.1:9650",
  }
}
```
